# Login & Registration System (Spring Boot)

A robust backend implementation of a user authentication system built with **Spring Boot 3.3.5** and **Java 21**. This project serves as a foundational component for a Personal Portfolio API, focusing on secure data persistence and RESTful service architecture.

## 🚀 Features

- **User Registration**: Securely add new users to the system.
- **User Login**: Validate user credentials against a persistent database.
- **RESTful API**: Clean and predictable URL structures for authentication.
- **Automated Documentation**: Integrated with Swagger/OpenAPI for real-time API testing and visualization.
- **Database Persistence**: Utilizes MySQL and Spring Data JPA for efficient data management.

## 🛠️ Tech Stack

- **Language**: Java 21
- **Framework**: Spring Boot 3.3.5
- **Build Tool**: Maven
- **Database**: MySQL
- **ORM**: Spring Data JPA
- **API Documentation**: SpringDoc OpenAPI (Swagger)
- **Developer Tools**: Spring Boot DevTools for rapid iteration

## 📋 Prerequisites

Before running this project, ensure you have:
- Java 21 or higher installed.
- MySQL Server installed and running.
- A basic understanding of Maven.
